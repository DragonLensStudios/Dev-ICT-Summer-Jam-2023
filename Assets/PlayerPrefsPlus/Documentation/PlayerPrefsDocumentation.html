PlayerPrefs+ Official Documentation

Initializing

To initialize, add the following line of code to the top of your script:

using PlayerPrefsPlus;

Storing Encrypted Variables

The following code will demonstrate how to encrypt PlayerPrefs.

void encryptString()

{

        //Encrypt A String With The Value Of

        //”Test String”

        PPPlus.SetStringEncrypted(“myKey”,“Test String”,

“MyPassword”);

//The Password Can Be Anything

}

void encryptInt()

{

        //Encrypt An Int With The Value Of

        //24

        PPPlus.SetIntEncrypted(“Highscore”,24,

“MyPassword”);

//The Key Is For Later Referral

//Example: Highscore

}

void encryptFloat()

{

        //Encrypt A Float With The Value Of

        //24.5

        PPPlus.SetFloatEncrypted(“Money”,24.5f,

“MyPassword”);

}

Decrypting Variables

The following code will demonstrate how to get data from an encrypted key.

void decryptString()

{

        //Decrypt A String With The Key Of

        //”myKey”

        var myString = PPPlus.GetStringEncrypted(“myKey”,

“MyPassword”);

//The Password Must Be The Same One That

//You Used When You Set The Key

//myString will now equal to “Test String”

}

void decryptInt()

{

        //Decrypt An Int With The Key Of

        //Highscore

        var myInt = PPPlus.GetIntEncrypted(“Highscore”,

“MyPassword”);

//myInt will now equal to 24

}

void decryptFloat()

{

        //Decrypt A Float With The Key Of

        //Money

        var myFloat = PPPlus.GetFloatEncrypted(“Money”,

“MyPassword”);

        //myFloat will now equal to 24.5

}

Storing Arrays In PlayerPrefs

The following code will demonstrate how to store an array in PlayerPrefs.

void storeStringArray()

{

//Stores The Cars Array In PlayerPrefs

        string[] cars = { “BMW”, “Audi”,“Mercedes” }

        PPPlus.SetStringArray(“Cars”, cars);

}

void storeIntArray()

{

//Stores The Car Years Array In PlayerPrefs

        int[] years = { 2017, 2019, 2021 }

        PPPlus.SetIntArray(“Years”, years);

}

void storeFloatArray()

{

//Stores The Car Engine Capacities In PlayerPrefs

        float[] engines = { 3.8f, 4.5f, 5.1f }

        PPPlus.SetFloatArray(“Engine”, engines);

}

Getting Arrays In PlayerPrefs

The following code will demonstrate how to get an array from PlayerPrefs.

void getCars()

{

         string[] cars = PPPlus.GetStringArray("Cars");

    int[] years = PPPlus.GetIntArray("Years");

    float[] engines = PPPlus.GetFloatArray("Engine");

}

Vector2 and Vector3 Values

The following code will demonstrate how to get and set Vector2/Vector3 Values.

void vector2()

{

         PPPlus.SetVector2("Position", transform.position);

    Vector2 position = PPPlus.GetVector2("Position");

}

void vector3()

{

         PPPlus.SetVector3("Pos3D", transform.position);

}

Other Commands

PPPlus.SetBool(“Key”, true); will store a boolean value in PlayerPrefs.

PPPlus.GetBool(“Key”); will get a boolean value from PlayerPrefs.

PPPlus.SetInt(“Key”, 1); will store an integer value in PlayerPrefs.

PPPlus.GetInt(“Key”); will get an integer value from PlayerPrefs.

PPPlus.SetFloat(“Key”, 1.5f); will store a float value in PlayerPrefs.

PPPlus.GetFloat(“Key”); will get a float value from PlayerPrefs.

PPPlus.SetString(“Key”, “A String”); will store a string in PlayerPrefs.

PPPlus.GetString(“Key”); will get a string from PlayerPrefs.

PPPlus.SetAnything<EpicObject>(MyObject, “myKey”); will store an object with class “EpicObject” and a name of “MyObject” in the PlayerPrefs key “myKey”. The EpicObject class HAS TO BE MARKED AS [System.Serializable] or else it will throw an exception.

PPPlus.ClearAllKeys(true); will clear all PlayerPrefs
